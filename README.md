# 🤖 InterviewNode: AI-Powered Technical Interview Simulator

InterviewNode is an intelligent, AI-driven platform designed to simulate a realistic technical interview experience for Data Science and Software Engineering candidates. It conducts interviews using context-aware questions tailored to the candidate's resume and target job role.

## ✨ Features

* **Interactive AI Interviewer:** Simulates a structured interview flow with dynamic, context-aware questions.
* **Drill-Down Logic:** Uses adaptive questioning to explore technical depth by following up on specific user answers.
* **Role-Specific Scenarios:** Tailored interview challenges based on job descriptions and individual resume context.
* **Communication Coaching:** Provides real-time feedback on English grammar, professional phrasing, and technical terminology.
* **UI:** Built with an intuitive graphical interface including custom animations for a seamless user experience.
* **Production Ready:** Optimized for fast, efficient performance and easy deployment via Hugging Face Spaces.

## 🛠️ Tech Stack

This project is built using the following technologies:

* **Frontend:** Streamlit
* **AI Backend:** LangChain, Groq API (Llama 3.3)
* **Deployment:** Docker, Hugging Face Spaces

## 🚀 Live Demo

You can try out the live application here: [https://huggingface.co/spaces/nimasha-w/ai-interviewer](https://huggingface.co/spaces/nimasha-w/ai-interviewer)

## ⚙️ Installation & Usage

* **Deployment:** This application is deployed on Hugging Face Spaces using the repository's main branch.
* **Environment Variables:** Ensure your `GROQ_API_KEY` is added to the Space's 'Secrets' configuration to enable AI functionalities.
